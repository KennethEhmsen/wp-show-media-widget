Display media images or pdf documents in a widget filtered by categories

== Description ==

This plugin is used to display thumbnails (from either images or pdf documents) as widget, filtered by the media category shipped.
It is dependent on the enhanced media library as it uses the taxonomy feature for filtering the media lbrary

Categories can by achieved by using the plugin "Enhanced Media Library" (https://wordpress.org/plugins/enhanced-media-library/)

== Installation ==
Add it through wordpress or unpack the downloaded zip file into your wp-content/plugins directory
